**Visual Regression**

* Use Playwright to navigate to `/` at **1440×900**, wait for fonts to load, then capture screenshots of:

  * Full dashboard
  * Financial Overview row
  * Market Intelligence column (left + right)
  * Data Analysis row (left + right cards)
* Compare using **jest-image-snapshot** with `failureThreshold: 0.02` (2% pixel diff) and `comparisonMethod: 'ssim'`.
* Include a “golden” baseline generated after first render that visually matches the provided screenshots (name them `baseline_dashboard.png`, etc.).
* Tests fail if diffs exceed threshold.

**Example Playwright test sketch (agent must implement fully):**

```ts
test('dashboard matches baseline', async ({ page }) => {
  await page.goto('http://localhost:5173');
  await page.setViewportSize({ width: 1440, height: 900 });
  await page.waitForSelector('[data-testid="financial-overview"]');
  const shot = await page.screenshot();
  expect(shot).toMatchImageSnapshot({
    customSnapshotIdentifier: 'baseline_dashboard',
    failureThreshold: 0.02,
    failureThresholdType: 'percent',
    comparisonMethod: 'ssim'
  });
});
```
